Netflix Content Analytics Dashboard
Overview

This project analyzes the Netflix Movies & TV Shows dataset using Power BI. It demonstrates the complete data analytics workflow, including data cleaning, transformation, data modeling, DAX calculations, and interactive dashboard development.

Problem Statement

Netflix contains thousands of movies and TV shows across different genres, countries, and ratings. This project aims to provide insights into:

Content distribution
Rating trends
Genre popularity
Country-wise content availability
Yearly content additions
Tools Used
Power BI
Power Query
DAX
Microsoft Excel

## Data Cleaning
Removed duplicate records
Trimmed extra spaces
Handled missing values
Normalized multi-value columns
Created relational tables

##Dashboard Features
KPI Cards
Movies vs TV Shows
Rating Distribution
Genre Analysis
Country Analysis
Release Year Trend
Interactive Filters
Drill-through
Content Explorer Page
